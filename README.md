# Website Scanner
A lightweight Python script to measure execution time and task duration.

## Features
* Precise timing using the `time` library.
* Ubuntu-compatible.

## How to Run
1. Clone the repo:
   `git clone https://github.com/YOUR_USER/YOUR_REPO.git`
2. Run the script:
   `python3 scanner.py`

## Upcoming Upgrades
